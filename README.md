# manaT

まなてぃ - Manager for Trpg characters' info

## これなぁに

TRPG 向けのキャラクターをはじめとするオブジェクトストアとして使います。

## コンセプト

### 元々の思想

あらゆるオンセツールはユーザが TRPG を楽しめるようにすること (= ユーザメリット) に注力して開発されるべきである

### その為に本アプリで実現したいこと

- ツールが持つキャラクター管理機能を容易に実装可能にする
- 既存のツールとの高い互換性と親和性。既存の資源を活かして実装可能にする

## どうやって起動するの

```
npm install
npm start
```
